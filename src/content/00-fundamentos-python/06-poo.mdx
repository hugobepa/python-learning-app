## 📚 Teoría Esencial

### Los 4 Pilares de la POO

| **Pilar**           | **Descripción**                          | **Ejemplo**             |
| ------------------- | ---------------------------------------- | ----------------------- |
| **Abstracción**     | Modelar objetos del mundo real           | `class Usuario:`        |
| **Encapsulamiento** | Ocultar datos internos                   | `self.__password`       |
| **Herencia**        | Reutilizar código de clase padre         | `class Admin(Usuario):` |
| **Polimorfismo**    | Mismo método, diferentes comportamientos | `def saludar():`        |

---

### Principios SOLID

| **Principio**             | **Descripción**                                   | **Beneficio**     |
| ------------------------- | ------------------------------------------------- | ----------------- |
| **S**ingle Responsibility | Una clase = una responsabilidad                   | Fácil de mantener |
| **O**pen/Closed           | Abierto para extensión, cerrado para modificación | Escalable         |
| **L**iskov Substitution   | Subclases sustituibles por clase padre            | Predecible        |
| **I**nterface Segregation | Interfaces pequeñas y específicas                 | Flexible          |
| **D**ependency Inversion  | Depender de abstracciones, no concreciones        | Desacoplado       |

## 💻 Ejemplo Guiado

### Ejemplo Junior

```python
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        print("Hola, soy " + self.nombre)

    def cumplir_anios(self):
        self.edad += 1

# Uso
persona = Persona("Hugo", 30)
persona.saludar()
persona.cumplir_anios()
```

### Ejemplo Senior

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass, field
from typing import List, Optional
from datetime import datetime

@dataclass
class Usuario(ABC):
    """Clase abstracta base para todos los usuarios."""
    id: int
    nombre: str
    email: str
    fecha_creacion: datetime = field(default_factory=datetime.now)

    @abstractmethod
    def obtener_permisos(self) -> List[str]:
        """Devuelve lista de permisos del usuario."""
        pass

    def saludar(self) -> str:
        return f"Hola, soy {self.nombre}"

@dataclass
class Admin(Usuario):
    """Administrador con permisos completos."""
    departamento: str = "IT"

    def obtener_permisos(self) -> List[str]:
        return ["leer", "escribir", "eliminar", "administrar"]

    def crear_usuario(self, nombre: str, email: str) -> Usuario:
        """Crea un nuevo usuario (solo admins)."""
        pass

@dataclass
class Viewer(Usuario):
    """Usuario solo lectura."""

    def obtener_permisos(self) -> List[str]:
        return ["leer"]

# Uso
if __name__ == "__main__":
    admin = Admin(id=1, nombre="Hugo", email="hugo@ejemplo.com")
    viewer = Viewer(id=2, nombre="Ana", email="ana@ejemplo.com")

    print(admin.saludar())
    print(f"Permisos admin: {admin.obtener_permisos()}")
    print(f"Permisos viewer: {viewer.obtener_permisos()}")
```

## 🎮 Ejercicio Práctico

**Reto:** Sistema de gestión de biblioteca

Crea un sistema con:

```markdown
1. Clase base `Libro` con título, autor, ISBN
2. Clase `LibroPrestado` que herede de `Libro` con fecha de préstamo
3. Clase `Usuario` que pueda prestar libros
4. Validación de que un usuario no preste más de 5 libros
```

-- POO: Programación Orientada a Objetos

```python
# Tu código aquí
from dataclasses import dataclass, field
from typing import List
from datetime import datetime

@dataclass
class Libro:
    titulo: str
    autor: str
    isbn: str
    disponible: bool = True

@dataclass
class LibroPrestado(Libro):
    fecha_prestamo: datetime = field(default_factory=datetime.now)
    fecha_devolucion: Optional[datetime] = None

@dataclass
class Usuario:
    nombre: str
    id: int
    libros_prestados: List[LibroPrestado] = field(default_factory=list)

    def prestar_libro(self, libro: Libro) -> bool:
        if len(self.libros_prestados) >= 5:
            print("Límite de libros alcanzado")
            return False
        if not libro.disponible:
            print("Libro no disponible")
            return False
        libro_prestado = LibroPrestado(
            titulo=libro.titulo,
            autor=libro.autor,
            isbn=libro.isbn,
            disponible=False
        )
        self.libros_prestados.append(libro_prestado)
        libro.disponible = False
        return True
```
