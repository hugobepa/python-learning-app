## 📚 Teoría Esencial

### Tipos de Datos Primitivos

| **Tipo**    | **Ejemplo**          | **Uso**           |
| ----------- | -------------------- | ----------------- |
| **`int`**   | `42`, `-10`, `0`     | Números enteros   |
| **`float`** | `3.14`, `-0.5`       | Números decimales |
| **`str`**   | `"hola"`, `'Python'` | Texto             |
| **`bool`**  | `True`, `False`      | Valores lógicos   |
| **`None`**  | `None`               | Ausencia de valor |

---

### Reglas de Nomenclatura

| **Regla**                  | **Válido**         | **Inválido**      |
| -------------------------- | ------------------ | ----------------- |
| **Sin espacios**           | `mi_variable`      | `mi variable`     |
| **Sin empezar con número** | `variable1`        | `1variable`       |
| **Usar guión bajo**        | `nombre_completo`  | `nombre-completo` |
| **Distingue mayúsculas**   | `Nombre`, `nombre` | -                 |

### Ejemplo Senior

```python
from typing import Final

PI: Final[float] = 3.14159

def calcular_imc(peso_kg: float, altura_m: float) -> float:
    """Calcula el Índice de Masa Corporal."""
    if altura_m <= 0:
        raise ValueError("La altura debe ser positiva")
    return round(peso_kg / (altura_m ** 2), 2)

if __name__ == "__main__":
    resultado = calcular_imc(72.0, 1.75)
    print(f"IMC: {resultado}")
```
