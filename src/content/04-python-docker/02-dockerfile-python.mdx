## 📚 Teoría Esencial

### Instrucciones Comunes de Dockerfile

| **Instrucción**        | **Propósito**                          | **Ejemplo** |
| ---------------------- | -------------------------------------- | ----------- |
| **FROM**               | Imagen base                            | `FROM python:3.12` |
| **WORKDIR**            | Directorio de trabajo                  | `WORKDIR /app` |
| **COPY**               | Copiar archivos                        | `COPY . .` |
| **RUN**                | Ejecutar comandos                      | `RUN pip install -r requirements.txt` |
| **CMD**                | Comando por defecto                    | `CMD ["python", "main.py"]` |
| **EXPOSE**             | Puerto expuesto                        | `EXPOSE 8000` |

---

### Imágenes Base de Python

| **Imagen**             | **Tamaño**                             | **Uso** |
| ---------------------- | -------------------------------------- | ------- |
| **python:3.12**        | ~1GB (completa)                        | Desarrollo |
| **python:3.12-slim**   | ~150MB (reducida)                      | Producción |
| **python:3.12-alpine** | ~50MB (mínima)                         | Producción optimizada |

## 💻 Ejemplo Guiado

### Dockerfile Básico para Python

```dockerfile
# Dockerfile
FROM python:3.12-slim

# Establecer directorio de trabajo
WORKDIR /app

# Copiar requirements primero (mejora cache)
COPY requirements.txt .

# Instalar dependencias
RUN pip install --no-cache-dir -r requirements.txt

# Copiar el resto del código
COPY . .

# Exponer puerto
EXPOSE 8000

# Comando para ejecutar la app
CMD ["python", "main.py"]
```

### Dockerfile Optimizado para FastAPI

```dockerfile
# Dockerfile
FROM python:3.12-slim

# Variables de entorno
ENV PYTHONDONTWRITEBYTECODE=1
ENV PYTHONUNBUFFERED=1

# Directorio de trabajo
WORKDIR /app

# Instalar dependencias
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copiar aplicación
COPY . .

# Usuario no root (seguridad)
RUN useradd -m -u 1000 appuser && chown -R appuser:appuser /app
USER appuser

# Exponer puerto
EXPOSE 8000

# Ejecutar con uvicorn
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### Comandos de Docker

```bash
# Construir imagen
docker build -t mi-app-python .

# Listar imágenes
docker images

# Ejecutar contenedor
docker run -p 8000:8000 mi-app-python

# Ejecutar en segundo plano
docker run -d -p 8000:8000 --name mi-app mi-app-python

# Ver logs
docker logs mi-app

# Detener contenedor
docker stop mi-app

# Eliminar contenedor
docker rm mi-app
```

## ✅ Check de Comprensión

**1. ¿Qué instrucción define la imagen base?**
````markdown
- `BASE`
- `FROM` ✅
- `IMAGE`
````
**2. ¿Qué imagen es más pequeña?**
````markdown
- `python:3.12`
- `python:3.12-slim` ✅
- `python:3.12-full`
````
**3. ¿Para qué sirve `--no-cache-dir` en pip?**
````markdown
- Acelera la instalación
- Reduce el tamaño de la imagen ✅
- Usa cache local
````
