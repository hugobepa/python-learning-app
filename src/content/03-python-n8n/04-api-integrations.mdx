## 📚 Teoría Esencial

### Tipos de Autenticación en APIs

| **Tipo**         | **Descripción**                   | **Ejemplo**            |
| ---------------- | --------------------------------- | ---------------------- |
| **API Key**      | Clave en header o query           | `X-API-Key: tu_key`    |
| **Bearer Token** | Token JWT en Authorization header | `Bearer <token>`       |
| **OAuth2**       | Flujo de autorización completo    | Google, GitHub         |
| **Basic Auth**   | Usuario y contraseña en base64    | `Authorization: Basic` |

---

## 💻 Ejemplo Guiado

### Configurar Credenciales en n8n

```bash
# 1. Ve a Credentials en n8n
# 2. Click en "Add Credential"
# 3. Selecciona el tipo (HTTP Header Auth, OAuth2, etc.)
# 4. Ingresa tus credenciales
# 5. Guarda y asigna al nodo HTTP Request
```

### HTTP Request a API con Auth

```json
{
  "nodes": [
    {
      "name": "HTTP Request",
      "type": "n8n-nodes-base.httpRequest",
      "parameters": {
        "method": "GET",
        "url": "https://api.openweathermap.org/data/2.5/weather",
        "authentication": "genericCredentialType",
        "genericAuthType": "httpHeaderAuth",
        "sendQuery": true,
        "queryParameters": {
          "parameters": [
            { "name": "q", "value": "Madrid" },
            { "name": "appid", "value": "{{ $credentials.apiKey }}" }
          ]
        }
      }
    }
  ]
}
```

### Procesar Respuesta con Python

```python
# Procesar respuesta de API del clima
import json

data = items[0]['json']

# Extraer datos relevantes
temperatura = data.get('main', {}).get('temp', 0) - 273.15  # Kelvin a Celsius
ciudad = data.get('name', 'Desconocida')
descripcion = data.get('weather', [{}])[0].get('description', '')

# Retornar formato personalizado
return [{
    "json": {
        "ciudad": ciudad,
        "temperatura_celsius": round(temperatura, 2),
        "condicion": descripcion,
        "timestamp": "2026-03-21"
    }
}]
```

### API con Paginación

```python
# Manejar paginación de API
import requests

all_items = []
page = 1

while True:
    response = requests.get(
        f"https://api.example.com/items",
        params={"page": page, "limit": 100},
        headers={"Authorization": "Bearer token"}
    )

    data = response.json()
    all_items.extend(data.get('items', []))

    if not data.get('has_more'):
        break
    page += 1

return [{"json": {"total": len(all_items), "items": all_items}}]
```

## 🎮 Ejercicio Práctico

**Reto:** Integración con API de GitHub

Crea un workflow que:

```markdown
1. Configure credenciales de GitHub (OAuth2 o token)
2. Obtenga tus repositorios mediante API
3. Procese los datos con Python
4. Filtre repositorios por lenguaje
5. Retorne lista formateada
```

-- fichero n8n HTTP Request --

````json
{
  "method": "GET",
  "url": "https://api.github.com/user/repos",
  "authentication": "genericCredentialType",
  "genericAuthType": "httpHeaderAuth",
  "sendQuery": false
}
```python
# Tu código aquí (nodo Code de n8n)
import json

# Obtener respuesta de GitHub API
repos = items[0]['json']

# Filtrar y procesar
python_repos = []
for repo in repos:
    if repo.get('language') == 'Python':
        python_repos.append({
            "nombre": repo.get('name'),
            "descripcion": repo.get('description', 'Sin descripción'),
            "stars": repo.get('stargazers_count', 0),
            "forks": repo.get('forks_count', 0),
            "url": repo.get('html_url')
        })

# Ordenar por stars
python_repos.sort(key=lambda x: x['stars'], reverse=True)

return [{"json": {
    "total_python_repos": len(python_repos),
    "repos": python_repos[:10]  # Top 10
}}]
````
