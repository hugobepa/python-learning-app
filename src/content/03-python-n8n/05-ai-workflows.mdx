## 💻 Ejemplo Guiado

### Configurar OpenAI en n8n

```bash
# 1. Obtén API Key de OpenAI
# https://platform.openai.com/api-keys

# 2. En n8n, ve a Credentials
# 3. Add Credential → OpenAI API
# 4. Ingresa tu API Key
# 5. Guarda y asigna al nodo OpenAI
```

### Workflow de Resumen de Texto

```json
{
  "nodes": [
    {
      "name": "OpenAI",
      "type": "@n8n/n8n-nodes-langchain.openAi",
      "parameters": {
        "operation": "text",
        "model": "gpt-3.5-turbo",
        "prompt": "Resume el siguiente texto en 3 puntos clave:\n\n{{ $json.texto }}",
        "maxTokens": 500
      }
    }
  ]
}
```

### Procesar con Python + IA

```python
# Procesar respuesta de OpenAI en n8n
import json

# Obtener respuesta de IA
respuesta = items[0]['json'].get('output', '')

# Extraer puntos clave
puntos = []
for linea in respuesta.split('\n'):
    if linea.strip().startswith(('1.', '2.', '3.', '-')):
        puntos.append(linea.strip())

# Retornar formato estructurado
return [{
    "json": {
        "resumen": respuesta,
        "puntos_clave": puntos,
        "longitud": len(respuesta),
        "procesado": True
    }
}]
```

### Workflow de Clasificación de Emails

```python
# Clasificar emails con IA
import json

prompt = f"""
Clasifica este email en una categoría:
- ventas
- soporte
- reclamacion
- otro

Email: {items[0]['json'].get('contenido', '')}

Responde solo con la categoría.
"""

# En n8n, esto iría en nodo OpenAI
# La respuesta se procesa aquí
categoria = items[0]['json'].get('ia_respuesta', 'otro').strip().lower()

return [{
    "json": {
        "email_original": items[0]['json'],
        "categoria": categoria,
        "prioridad": "alta" if categoria == "reclamacion" else "normal",
        "timestamp": "2026-03-21"
    }
}]
```

## 🎮 Ejercicio Práctico

**Reto:** Crea un workflow de análisis de sentimiento
````markdown
1. Configura credenciales de OpenAI en n8n
2. Crea webhook para recibir textos
3. Añade nodo OpenAI para análisis
4. Procesa respuesta con Python
5. Retorna sentimiento y puntuación
````

-- nodo Code de n8n --
```python
# Tu código aquí (nodo Code de n8n)
import json

# Obtener respuesta de IA
respuesta_ia = items[0]['json'].get('output', '')

# Parsear sentimiento
sentimiento_map = {
    "positivo": "😊",
    "negativo": "😞",
    "neutral": "😐"
}

# Extraer sentimiento (asumiendo formato de IA)
sentimiento = "neutral"
if "positivo" in respuesta_ia.lower():
    sentimiento = "positivo"
elif "negativo" in respuesta_ia.lower():
    sentimiento = "negativo"

return [{
    "json": {
        "texto_original": items[0]['json'].get('texto', ''),
        "sentimiento": sentimiento,
        "emoji": sentimiento_map.get(sentimiento, "😐"),
        "analisis": respuesta_ia,
        "procesado": True
    }
}]
```
