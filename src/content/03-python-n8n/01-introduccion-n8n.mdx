## 📚 Teoría Esencial

### ¿Qué es n8n?

| **Característica**     | **Descripción**                     | **Beneficio**       |
| ---------------------- | ----------------------------------- | ------------------- |
| **Workflow-based**     | Automatización visual con nodos     | Sin código complejo |
| **Self-hostable**      | Puedes alojarlo en tu servidor      | Control total       |
| **Python Integration** | Ejecuta código Python personalizado | Flexibilidad        |
| **500+ Integrations**  | Conecta con APIs, DBs, servicios    | Ecosistema amplio   |

## 💻 Ejemplo Guiado

### Instalación de n8n

```bash
# Opción 1: npm (recomendado para desarrollo)
npm install n8n -g
n8n start

# Opción 2: Docker
docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n

# Opción 3: n8n Cloud (sin instalación)
# https://n8n.io/cloud
```


### Primer Workflow Básico

```json
{
  "nodes": [
    {
      "name": "Start",
      "type": "n8n-nodes-base.manualTrigger",
      "position": [250, 300]
    },
    {
      "name": "Python Code",
      "type": "n8n-nodes-base.python",
      "position": [450, 300],
      "parameters": {
        "code": "return [{\"json\": {\"message\": \"Hola desde Python!\"}}]"
      }
    }
  ],
  "connections": {
    "Start": {
      "main": [[{ "node": "Python Code", "type": "main", "index": 0 }]]
    }
  }
}
```

### Acceder a n8n

```bash
# Después de iniciar n8n
# Abre en navegador: http://localhost:5678

# Credenciales por defecto:
# - Email: tu@email.com
# - Password: tu_contraseña
```

## 🎮 Ejercicio Práctico

**Reto:** Crea tu primer workflow en n8n
````markdown
1. Instala n8n localmente o usa n8n Cloud
2. Crea un nuevo workflow
3. Añade un nodo "Manual Trigger"
4. Añade un nodo "Code" con Python
5. Ejecuta el workflow y verifica el output
````

-- python en nodo Code de n8n:
```python
# Tu código aquí (nodo Code de n8n)
items = []
for i in range(3):
    items.append({
        "json": {
            "id": i + 1,
            "message": f"Mensaje número {i + 1}",
            "timestamp": "2026-03-21"
        }
    })
return items
```
