## 📚 Teoría Esencial

### Componentes del Frontend

| **Componente** | **Propósito**             | **Ubicación**     |
| -------------- | ------------------------- | ----------------- |
| **Pages**      | Páginas de la aplicación  | `src/pages/`      |
| **Layouts**    | Plantillas reutilizables  | `src/layouts/`    |
| **Components** | Componentes UI            | `src/components/` |
| **Content**    | Contenido MDX (lecciones) | `src/content/`    |
| **Styles**     | Estilos CSS               | `src/styles/`     |

---

### Integración Frontend-Backend

| **Endpoint**                     | **Propósito**            | **Método** |
| -------------------------------- | ------------------------ | ---------- |
| `GET /`                          | Página de inicio         | GET        |
| `GET /modules`                   | Listar todos los módulos | GET        |
| `GET /modules/[module]`          | Lecciones de un módulo   | GET        |
| `GET /modules/[module]/[lesson]` | Contenido de lección     | GET        |
| `GET /health`                    | Health check del backend | GET        |

## 💻 Ejemplo Guiado

### Configurar Astro para API

```javascript
// astro.config.mjs
import { defineConfig } from "astro/config";

export default defineConfig({
  site: "https://python-learning-app.com",
  output: "static",
  server: {
    port: 4321,
    host: true,
  },
});
```

### Página de Lección Dinámica (src/pages/modules/[module]/[lesson].astro)

```astro
---
import BaseLayout from '../../../layouts/BaseLayout.astro';

export async function getStaticPaths() {
  const response = await fetch('http://localhost:8000/api/lessons');
  const lessons = await response.json();

  return lessons.map((lesson) => ({
    params: {
      module: lesson.module,
      lesson: lesson.order.toString()
    },
    props: { lesson }
  }));
}

const { lesson } = Astro.props;
---

<BaseLayout title={lesson.title}>
  <article>
    <h1>{lesson.title}</h1>
    <p class="meta">
      <span>⏱️ {lesson.duration}</span>
      <span>⭐ {lesson.xp} XP</span>
    </p>
    <div class="content">
      <slot />
    </div>
  </article>
</BaseLayout>

<style>
  .meta {
    display: flex;
    gap: 1rem;
    color: #666;
  }
  .content {
    background: white;
    padding: 2rem;
    border-radius: 8px;
    margin-top: 1rem;
  }
</style>
```

## ✅ Check de Comprensión

**1. ¿Dónde se almacenan los layouts en Astro?**

```markdown
- `src/layouts/` ✅
- `src/templates/`
- `src/views/`
```

**2. ¿Qué función define rutas dinámicas?**

```markdown
- `getStaticPaths()` ✅
- `getDynamicRoutes()`
- `definePaths()`
```

**3. ¿Cómo se importan componentes en Astro?**

```markdown
- `import Component from '../components/Component.astro'` ✅
- `require('../components/Component.astro')`
- `include '../components/Component.astro'`
```
