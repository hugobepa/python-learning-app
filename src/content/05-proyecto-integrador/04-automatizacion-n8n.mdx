## 📚 Teoría Esencial

### Workflows del Proyecto Integrador

| **Workflow**       | **Propósito**              | **Trigger** |
| ------------------ | -------------------------- | ----------- |
| **Notificaciones** | Enviar emails de progreso  | Webhook     |
| **Reportes**       | Generar reportes semanales | Schedule    |
| **IA Analysis**    | Analizar progreso con IA   | Manual      |
| **Backup**         | Backup automático de datos | Schedule    |

---

### Integración n8n + FastAPI

| **Endpoint**                  | **Propósito**               | **Método** |
| ----------------------------- | --------------------------- | ---------- |
| `POST /api/webhooks/progress` | Recibir progreso de usuario | POST       |
| `GET /api/users/:id/progress` | Obtener progreso de usuario | GET        |
| `POST /api/notifications`     | Enviar notificación         | POST       |
| `GET /api/reports/weekly`     | Generar reporte semanal     | GET        |

## 💻 Ejemplo Guiado

### Workflow de Notificaciones de Progreso

```json
{
  "nodes": [
    {
      "name": "Webhook",
      "type": "n8n-nodes-base.webhook",
      "parameters": {
        "httpMethod": "POST",
        "path": "user-progress",
        "responseMode": "lastNode"
      }
    },
    {
      "name": "OpenAI",
      "type": "@n8n/n8n-nodes-langchain.openAi",
      "parameters": {
        "operation": "text",
        "model": "gpt-3.5-turbo",
        "prompt": "Genera un mensaje motivacional para un usuario que completó {{ $json.lesson }} con {{ $json.xp }} XP"
      }
    },
    {
      "name": "Email",
      "type": "n8n-nodes-base.emailSend",
      "parameters": {
        "to": "{{ $json.user_email }}",
        "subject": "¡Felicidades por tu progreso!",
        "body": "{{ $json.output }}"
      }
    }
  ],
  "connections": {
    "Webhook": {
      "main": [[{ "node": "OpenAI", "type": "main", "index": 0 }]]
    },
    "OpenAI": {
      "main": [[{ "node": "Email", "type": "main", "index": 0 }]]
    }
  }
}
```

### Workflow de Reportes Semanales

```json
{
  "nodes": [
    {
      "name": "Schedule",
      "type": "n8n-nodes-base.scheduleTrigger",
      "parameters": {
        "rule": {
          "interval": [
            {
              "field": "days",
              "hours": 9,
              "minutes": 0
            }
          ]
        }
      }
    },
    {
      "name": "HTTP Request",
      "type": "n8n-nodes-base.httpRequest",
      "parameters": {
        "method": "GET",
        "url": "http://backend:8000/api/reports/weekly"
      }
    },
    {
      "name": "Slack",
      "type": "n8n-nodes-base.slack",
      "parameters": {
        "channel": "#progreso",
        "text": "Reporte semanal: {{ $json.total_xp }} XP esta semana"
      }
    }
  ]
}
```

### Configurar n8n en Docker Compose

```yaml
# docker-compose.yml (sección n8n)
n8n:
  image: n8nio/n8n:latest
  container_name: n8n-automation
  ports:
    - "5678:5678"
  environment:
    - N8N_HOST=localhost
    - N8N_PORT=5678
    - WEBHOOK_URL=http://localhost:5678/
    - N8N_BASIC_AUTH_ACTIVE=true
    - N8N_BASIC_AUTH_USER=admin
    - N8N_BASIC_AUTH_PASSWORD=${N8N_PASSWORD}
  volumes:
    - n8n-data:/home/node/.n8n
  networks:
    - app-network
  restart: always
```

```markdown
5. Prueba el workflow completo
```

-archivo cofiguración n8n:

```yaml
# Tu código aquí
# docker-compose.yml (n8n section)
n8n:
  image: n8nio/n8n:latest
  ports:
    - "5678:5678"
  environment:
    - N8N_HOST=localhost
    - WEBHOOK_URL=http://localhost:5678/
  volumes:
    - n8n-data:/home/node/.n8n
  networks:
    - app-network
```

-- webhooks:

```python
# Backend endpoint para trigger de n8n
# backend/app/routes/webhooks.py
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel

router = APIRouter()

class ProgressData(BaseModel):
    user_id: int
    lesson: str
    xp: int
    user_email: str

@router.post("/webhooks/progress")
async def user_progress(data: ProgressData):
    # Trigger n8n webhook
    import requests
    requests.post(
        "http://n8n:5678/webhook/user-progress",
        json=data.dict()
    )
    return {"status": "notification sent"}
```

-- comandos:

```bash
# Comandos para probar
# Iniciar n8n
docker-compose up -d n8n

# Acceder a n8n
# http://localhost:5678

# Probar webhook
curl -X POST http://localhost:8000/api/webhooks/progress \
  -H "Content-Type: application/json" \
  -d '{"user_id": 1, "lesson": "Python Basics", "xp": 50, "user_email": "user@example.com"}'
```
