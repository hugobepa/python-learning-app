## 📚 Teoría Esencial

### Operaciones CRUD

| **Operación** | **Método HTTP** | **Endpoint**  | **Status Code** |
| ------------- | --------------- | ------------- | --------------- |
| **Create**    | POST            | `/items/`     | 201 Created     |
| **Read**      | GET             | `/items/{id}` | 200 OK          |
| **Update**    | PUT/PATCH       | `/items/{id}` | 200 OK          |
| **Delete**    | DELETE          | `/items/{id}` | 204 No Content  |

---

### HTTP Status Codes

| **Código** | **Significado**       | **Cuándo Usar**          |
| ---------- | --------------------- | ------------------------ |
| **200**    | OK                    | GET, PUT, PATCH exitosos |
| **201**    | Created               | POST exitoso             |
| **204**    | No Content            | DELETE exitoso           |
| **400**    | Bad Request           | Datos inválidos          |
| **404**    | Not Found             | Recurso no existe        |
| **500**    | Internal Server Error | Error del servidor       |

## 💻 Ejemplo Guiado

### CRUD Completo

```python
from fastapi import FastAPI, HTTPException, status
from pydantic import BaseModel
from typing import Optional, List

app = FastAPI()

class Item(BaseModel):
    id: int
    nombre: str
    precio: float
    descripcion: Optional[str] = None

# Base de datos simulada
items_db = {}

@app.post("/items/", response_model=Item, status_code=status.HTTP_201_CREATED)
def crear_item(item: Item):
    if item.id in items_db:
        raise HTTPException(status_code=400, detail="El ID ya existe")
    items_db[item.id] = item
    return item

@app.get("/items/", response_model=List[Item])
def listar_items():
    return list(items_db.values())

@app.get("/items/{item_id}", response_model=Item)
def obtener_item(item_id: int):
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="Item no encontrado")
    return items_db[item_id]

@app.put("/items/{item_id}", response_model=Item)
def actualizar_item(item_id: int, item: Item):
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="Item no encontrado")
    items_db[item_id] = item
    return item

@app.delete("/items/{item_id}", status_code=status.HTTP_204_NO_CONTENT)
def eliminar_item(item_id: int):
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="Item no encontrado")
    del items_db[item_id]
    return None
```
