## 📚 Teoría Esencial

### ¿Qué es FastAPI?

| **Característica**   | **Descripción**                | **Beneficio**    |
| -------------------- | ------------------------------ | ---------------- |
| **Rápido**           | Basado en Starlette y Pydantic | Alto rendimiento |
| **Tipo seguro**      | Type hints de Python           | Menos bugs       |
| **Auto-documentado** | Swagger UI automático          | Fácil testing    |
| **Asíncrono**        | Soporte nativo async/await     | Escalable        |

---

### FastAPI vs Otros Frameworks

| **Framework** | **Velocidad**   | **Type Hints** | **Docs Auto** | **Curva** |
| ------------- | --------------- | -------------- | ------------- | --------- |
| **FastAPI**   | ⚡⚡ Muy rápido | ✅ Sí          | ✅ Sí         | Baja      |
| **Flask**     | ⚡⚡ Rápido     | ❌ No          | ❌ No         | Baja      |
| **Django**    | ⚡ Moderado     | ⚠️ Parcial     | ❌ No         | Alta      |

## 💻 Ejemplo Guiado

### Instalación

```bash
# Crear entorno virtual
python -m venv .venv

# Activar (Windows)
.venv\Scripts\Activate.ps1

# Activar (macOS/Linux)
source .venv/bin/activate

# Instalar FastAPI y Uvicorn
pip install fastapi uvicorn[standard]
```

### Primer Endpoint

```python
# main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hola Mundo"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}
```

### Ejecutar Servidor

```bash
# Iniciar servidor de desarrollo
uvicorn main:app --reload

# Acceder a:
# - API: http://localhost:8000
# - Docs: http://localhost:8000/docs
# - ReDoc: http://localhost:8000/redoc
```

## 🎮 Ejercicio Práctico

**Reto:** Crea tu primera API

```markdown
1. Crea una carpeta `mi-api-fastapi`
2. Crea un entorno virtual e instala FastAPI
3. Crea `main.py` con 3 endpoints:
   - `GET /` → Mensaje de bienvenida
   - `GET /saludo/{nombre}` → Saludo personalizado
   - `GET /operacion/{a}/{b}` → Suma de dos números
4. Ejecuta y prueba en el navegador
```

-- api.py --:

```python
# Tu código aquí
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def root():
    return {"message": "Bienvenido a mi API"}

@app.get("/saludo/{nombre}")
def saludar(nombre: str):
    return {"saludo": f"Hola {nombre}!"}

@app.get("/operacion/{a}/{b}")
def sumar(a: int, b: int):
    return {"resultado": a + b}
```
