## 🚀 Características

- **6 Módulos Completos**: Fundamentos, FastAPI, GitHub Actions, n8n, Docker, Proyecto Integrador
- **36 Lecciones**: Teoría concisa + ejemplos junior vs senior
- **100+ Ejercicios**: Validación automática de código
- **Gamificación**: XP, niveles, badges, rachas diarias
- **100% Responsive**: Optimizado para móvil y desktop
- **Accesible**: WCAG 2.1 AA compliant
- **PWA Ready**: Instalable como app nativa

## 🚀 Quick Start

### Prerrequisitos

- Node.js 18+
- npm

### Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/python-learning-app.git
cd python-learning-app

# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build producción
npm run build

# Preview producción
npm run preview
